6.ORM (Object-Relational Mapping):  
A technique for mapping data types and querying data more efficiently. It is part of Java EE but can also be used in Java SE via JPA.
EX: 
INSERT IN TO table_name VALUES(..., ...., ....,)
 through JPA:
SinhvienDAO.save(ojb) hoac saveorupdate(ojb)

7.Hibernate:  
Hibernate is a Java library that implements all JPA requirements. It is a framework that facilitates database interaction. With Hibernate, tables can be created quickly, and queries can be executed easily.
